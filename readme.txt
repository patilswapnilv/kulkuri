Fantastic One Page Theme!

== Description ==

Kulkuri is great one page theme. You can literally do everything in just one page and scroll to your content using beautiful fixed navigation.

== Installation ==

1. Upload `kulkuri` to the `/wp-content/themes/` directory.
1. Activate the theme through the 'Themes' menu in WordPress.
